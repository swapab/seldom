# Seldom

Seldom is a part of fake html server. Given an html selector it will generate a random fake HTML.

## Installation

Add this line to your application's Gemfile:

    gem 'seldom'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install seldom
